Getting Started with HTML & CSS

Building my first website was an exciting experience. I started with learning HTML to structure my pages, then used CSS to add colors, layouts, and fonts. I experimented with headers, paragraphs, lists, images, and links. It was amazing to see my first page come to life!

Through this project, I also learned about responsive design so my site looks good on mobile and desktop. Using developer tools in the browser helped me debug and improve my layout.
